Code Editor

For our code editor, we will be using Visual Studio Code, which is a free (open source) code editor
created and maintained by Microsoft. It also works on Windows, macOS, and Linux. Make sure you
have downloaded and installed it on all the computers you will use for web development.

Web Client

For our web client we will use the many web browsers we introduced in Week 1, namely:
  • Google Chrome for desktop and Android
  • Microsoft Edge and Internet Explorer (IE)
  • Apple Safari and Safari for iOS
  • Mozilla Firefox
  • Opera

Web Server

We will also need a web server to host our web pages and applications. Installing and running a web
server can be complicated.

For our purposes, we will use a simple node.js based http-server. Now start the web server by
running the following command: npx http-server.

http://127.0.0.1:8080 means connect using HTTP to my local computer on port 8080.

When you are done testing your web site, stop the web server by pressing CTRL-C in your terminal

To run the server again, use npx http-server.

NOTE: the second IP address will be different than the above, but 127.0.0.1 will always be correct.

Basic HTML5 Document

Here’s a basic HTML5 web page: 
  <!doctype html>
  <html>
      <head>
          <meta charset="utf-8">
          <title>My Web Page</title>
      </head>
      <body>
          <!-- This is a comment -->
          <h1>Hello World!</h1>
      </body>
  </html>
Every time we update anything in our web page, we have to refresh the web page in our browser. The web
server will serve the most recent version of the file on disk when it is requested.
